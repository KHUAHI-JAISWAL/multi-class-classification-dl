# Multi-Class Classification using Deep Learning (MNIST)

This project performs multi-class image classification on the MNIST dataset
using a Deep Learning model.

## Dataset
MNIST handwritten digits dataset (0 to 9)

## Problem Type
Multi-class classification

## Model
Neural Network implemented using Deep Learning

## File
mnist-classification.ipynb

## How to run
Open the notebook and run all cells in order.
